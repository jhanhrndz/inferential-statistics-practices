#  Inferential Statistics in R

This repository contains practical exercises developed in **R**, focused on real-world application of inferential statistics.

##  Topics

* Hypothesis testing.
* ANOVA.
* Linear regression
* Statistical assumptions validation

##  Approach

Each exercise emphasizes **statistical interpretation**, **correct test selection**, and clean implementation in R.

---

Ideal for practice, learning, and building a strong data analysis portfolio.
